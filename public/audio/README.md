# Audio Files for Aristotle Class Memories

## How to Add Your Music

1. Download the MP3 files for each student's theme song
2. Rename them according to the naming convention below
3. Place them in this `/public/audio/` folder
4. The website will automatically play the correct song when each profile is opened
